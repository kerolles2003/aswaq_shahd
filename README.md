# Aswaq Shahd Digital Magazine

A premium, mobile-first digital catalog for Aswaq Shahd Supermarket.

## Features

- **RTL & Arabic Support**: Fully localized interface.
- **Digital Magazine**: Browse offers page by page.
- **Product Search**: Fast instant search for products.
- **Categories**: Filter by Food, Fresh, Drinks, Cleaning.
- **Responsive Design**: Looks great on Mobile, Tablet, and Desktop.
- **Glassmorphism UI**: Modern, clean, and vibrant design.

## How to Run

### Option 1: Direct Open

Simply double-click `index.html` to open it in your default web browser.

### Option 2: Local Server (Recommended for best experience)

If you have Python installed:

```bash
python -m http.server
```

Then visit `http://localhost:8000`

## Project Structure

- `index.html`: Main entry point.
- `css/`: Stylesheets.
- `js/`: Application logic and data.
- `assets/img/`: Generated high-quality assets.
